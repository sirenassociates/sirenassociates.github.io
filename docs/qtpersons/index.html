Quarantine Persons

Quarantine Persons Upload Process

To upload the list of quarantined persons, there are a few steps you need to go through.

Step 1

Giving Permissions

In the Groups settings page, you pick the user whose permissions you want to change, click on the button "مراكز الحجر الصحي", and make sure the checkbox "تحميل بيانات الأشخاص المحجورين" is ticked.

Once enabled, the upload option "تحميل بيانات الأشخاص المحجورين" will appear in the navigation menu page under the "مراكز الحجر الصحي" title.

Step 2

Importing the Excel File

Once clicked, the upload popup window will open, and you can now click on the upload button to browse files on your computer and select the file you want to import.

Formatting Explanation
A brief explanation on how the data is entered:

Full Name is split into first, father, and last name.

Date format is dd/MM/yyyy or dd-MM-yyyy.

Time format is HH:mm.

Non existing lookups (the dropdown lists like municipality) are kept empty.

Pending: We will add the option to download the template from the upload popup.

Step 3

Entering the Data

No validation is done on the data.

Instead, we show the uploaded info in an editable table for the user to review, fix, and save.

Upon saving, APP/SMS notifications are sent to Municipality presidents.
